Sobre a Turma

Somos a turma de 1º ano de Desenvolvimento de Sistemas de 2024, do Colégio Estadual Cívico-Militar Tancredo de Almeida Neves, localizado na Rua José Carlos Pace, 1866, no bairro Parque Residencial Morumbi II, em Foz do Iguaçu, PR. Aqui, vivemos uma experiência única, integrando o ensino médio ao curso técnico, que nos permite aprender conteúdos essenciais e, ao mesmo tempo, explorar o mundo da tecnologia e da programação.

Nosso ambiente de estudo, além de oferecer uma estrutura que atende às nossas necessidades, nos proporciona um espaço onde trabalhamos juntos para desenvolver habilidades em lógica computacional, programação e desenvolvimento de software. Orientados por professores dedicados, encaramos desafios reais, colocando em prática os conceitos que aprendemos e crescendo como futuros desenvolvedores.

Aqui, não somos apenas colegas de turma. Somos um grupo que compartilha o desejo de aprender, inovar e colaborar, com o sonho de transformar ideias em soluções tecnológicas. O Colégio Tancredo de Almeida Neves se tornou nosso ponto de partida para o futuro, onde cultivamos tanto o conhecimento técnico quanto os valores de disciplina, trabalho em equipe e compromisso com o aprendizado.

Projeto 1: Tabuada Personalizada

Digite um número para ver sua tabuada.

function gerarTabuada() {
    let tabuadaDiv = document.getElementById("tabuada");
    tabuadaDiv.innerHTML = "";
    let numero = parseInt(document.getElementById("numero").value);
    if (!isNaN(numero)) {
        for (let i = 1; i <= 10; i++) {
            tabuadaDiv.innerHTML += `${numero} x ${i} = ${numero * i}
`;
        }
    } else {
        tabuadaDiv.innerHTML = "Por favor, insira um número válido.";
    }
}

function calcularMedia() {
    let trimestres = document.getElementById("trimestres").value;
    let somaNotas = 0;
    
    for (let i = 1; i <= trimestres; i++) {
        let nota = parseFloat(document.getElementById("nota" + i).value);
        if (!isNaN(nota)) {
            somaNotas += nota;
        }
    }
    
    let media = somaNotas / trimestres;
    document.getElementById("resultadoMedia").innerHTML = `A média é: ${media.toFixed(2)}`;
}

Nossos Professores

Professor Teotonio

Teotonio é o professor responsável pela disciplina de Lógica Computacional no primeiro ano de Desenvolvimento de Sistemas do curso técnico do ensino médio. Sua missão é ensinar aos alunos os fundamentos essenciais da lógica que sustentam o raciocínio computacional. Ele prepara os estudantes para desenvolverem habilidades em análise de problemas e elaboração de soluções baseadas em lógica matemática, algo crucial para a programação de sistemas e a construção de algoritmos eficientes. Com uma abordagem focada no pensamento crítico, Teotonio incentiva seus alunos a pensar de forma estruturada e a aplicar esses conceitos em desafios práticos, proporcionando uma base sólida para as disciplinas mais avançadas no curso.

Professor Marcelo

Marcelo leciona as disciplinas de Análise e Projeto de Sistemas e Ciência da Computação no primeiro ano de Desenvolvimento de Sistemas do curso técnico de ensino médio, com um enfoque abrangente tanto em hardware quanto em software. Ele não só ensina os conceitos teóricos e práticas de desenvolvimento de sistemas, mas também compartilha seu vasto conhecimento em hardware. Marcelo orienta seus alunos na montagem de computadores, identificação e instalação de peças, além de como formatar sistemas operacionais, especialmente o Windows, e realizar a instalação completa do sistema. Sua abordagem prática e teórica permite que os estudantes se familiarizem com todos os aspectos do funcionamento de um computador, preparando-os para enfrentar desafios técnicos no mundo da informática, tanto no nível de software quanto no de hardware. Essa combinação de habilidades os capacita a serem profissionais completos e prontos para atuar nas diversas áreas da Tecnologia da Informação.
